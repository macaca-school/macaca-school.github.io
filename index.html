HOW TO USE:

  1. Set a Time Interval.
  2. Set a Rhythm Breaker if you want to add a random extra time delay between each instruction. For example, if you set 1.0s as Time Interval and 50% as Rhythm Breaker then instructions will appear between 1.0s and 1.5s.
  3. Select the Instructions you want.
  4. Turn on audio on your device, press START and have fun!

* You don't need to pause to change any configuration. Just change it and the gadget will automatically pick it up for you.
